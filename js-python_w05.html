Koty Ali¶

  • Pierwszego dnia Ala dostaa jednego kota. Kadego kolejnego dnia dostaje o jednego wicej ni dnia poprzedniego. Ile kotw ma Ala po n dniach?
    • pierwszy dzie: 1
    • drugi dzie: 1 + 1
    • trzeci dzie: 1 + 1 + 1 ...

Koty Ali - algorytm I¶

In [8]:
def licz_koty_v1(n):
    """Zwraca liczb kotw po n dniach."""
    n_cats = 0                     # Ala nie ma kota
    for dzien in range(1, n + 1):  # ptla po dniach
        for koty in range(dzien):  # liczba kotw = nr dnia
            n_cats += 1            # dodaj kota
    return n_cats                  # dwie ptle -> n*n operacji

Wartość wielomianu¶

  • Dany jest wielomian:
$$ W(x) = a_0 + a_1\cdot x + a_2 \cdot x^2 + ... + a_n \cdot x^n$$
  • Korzystajc bezporednio z powyszgo wzoru naley wykona $n$ dodawa oraz $1 + 2 + ... + n = \frac{n\cdot(n+1)}{2}$ mnoe

Metoda naiwna¶

def naive(a, x):
    """Zwraca warto wielomianu o wspczynnikach a w punkcie x."""
    result = 0
    for i, ai in enumerate(a): # i - indeks, ai - wspczynnik a_i
        result += ai * x**i    # W(x) += a_i * x^i
    return result

Schemat Hornera¶

  • Zapisujc wielomian w postaci:
$$W(x) = a_0 + x\cdot(a_1 + x\cdot(a_2 + ... + x\cdot(a_{n-1} + a_n\cdot x)...)$$
  • organiczamy si do $n$ doda i $n$ mnoe

Algorytm Hornera¶

$$W(x) = a_0 + x\cdot(a_1 + x\cdot(a_2 + ... + x\cdot(a_{n-1} + a_n\cdot x)...)$$
  • Niech
$$b_n = a_n$$$$b_{n-1} = b_n\cdot x + a_{n-1}$$$$b_{n-2} = n_{n-1}\cdot x + a_{n-2}$$$$...$$$$b_1 = b_2\cdot x + a_1$$$$b_0 = b_1\cdot x + a_0$$
  • $b_0$ - warto wielomianu

def horner(a, x):
    """Zwraca warto wielomianu o wspczynnikach a w punkcie x."""
    result = 0
    for w in a[-1::-1]:         # "reverse"
        result = result * x + w # b_i = b_{i-1} * x + a_i
    return result

    """Drukuje tabel z procentowym wkadem kadego czonu."""
    table_head(f, width)

            print_cell(round(term / f_val * 100), width)
        print()

Notacja duego Ο

  • Funkcja $f(n)$ jest $O(g(n))$, jeli istniej liczby dodatnie $c$ i $n_0$, takie e:
$$\forall_{n>n_0} ~~~~~ f(n) < c\cdot g(n)$$
  • Jeli $f_1(n) \in O(g(n))$ i $f_2(n) \in O(g(n))$, to:
$$f_1(n) + f_2(n) \in O(g(n))$$$$f_1(n) \cdot f_2(n) \in O(g^2(n))$$

In [43]:
def quadratic(n):
    result = 0
    for i in range(n):
        for j in range(i, n):
            result += 1
    return result
  • ptla zewntrzna -> $n$ razy
  • ptla wewntrzna -> $(n-i)$ razy
  • w sumie $1 + 2 + ... + n = \frac{n\cdot(n + 1)}{2} = \frac{1}{2}n^2 + \frac{1}{2}n$
  • zoono asymptotyczna: $O(n^2) + O(n) = O(n^2)$

Przykłady rzędów złożności - $O(\log n)$¶

In [44]:
def logarithmic(n):
    result = 0
    while n > 1:
        n //= 2
        result += 1
    return result
  • w kadej iteracji zmniejszamy $n$ dwukrotnie
  • jeli $n = 2^x$, to wykonamy $\log n$ operacji
  • algorytm jest $O(\log n)$

Sortowanie bąbelkowe (bubble sort)¶

  1. porwnuj cyklicznie dwa ssiadujce elementy
  2. zmie kolejno jeli zaburzaj porzdek
  3. przerwij, jeli nie ma co zmienia

Sortowanie bąbelkowe - przykład¶

[3, 6, 4, 2] --> [3, 6, 4, 2] --> [3, 4, 6, 2] --> [3, 4, 2, 6]

[3, 4, 2, 6] --> [3, 4, 2, 6] --> [3, 2, 4, 6] --> [3, 2, 4, 6]

[3, 2, 4, 6] --> [2, 3, 4, 6] --> [2, 3, 4, 6] --> [2, 3, 4, 6]

[2, 3, 4, 6] --> [2, 3, 4, 6] --> [2, 3, 4, 6] --> [2, 3, 4, 6]

(brak zmian -> koniec)

Implementacja¶

In [48]:
def bubblesort(lst):
    """Sortuje podan list."""
    n = len(lst) # liczba elementw
    while True:  # powtarzaj do break
        swapped = False # ustaw flag na fasz
        for i in range(n-1): # ptla od 0 do n-2
            if lst[i] > lst[i+1]: # porzdek zaburzony
                lst[i], lst[i+1] = lst[i+1], lst[i] # podmie
                swapped = True # ustaw flag na prawd
        if not swapped: # przerwij jeli nie byo zmian
            break

def bubblesort_v3(lst):
    """Sortuje podan list."""
    n = len(lst) # liczba elementw
    while n > 0:
        new_n = 0 # miejsce ostatniej zmiany
        print(lst, end=' ')

Złożoność bubblesort¶

  • $n$ operacji -> co najmniej ostatni na swoim miejscu
  • $(n-1)$ operacji -> dwa ostatnie na swoim miejscu
  • $(n-2)$ operacji -> trzy na swoim miejscu itd...
  • $n + (n-1) + (n-2) + ... = O(n^2)$

Sortowanie przez scalanie (mergesort)¶

  • podziel nieposortowan list na $n$ "podlist" zawierajcych po jednym elemencie
  • cyklicznie scalaj "podlisty" w posortowane "podlisty", a zostanie jedna

Start: [3, 6, 4, 2]

Split: [3] [6] [4] [2]

Merge: [3, 6] [2, 4]

Merge: [2, 3, 4, 6]

Algorytm scalania¶

  • majc dwie uporzdkowane listy
  • porwnaj skrajne elementy i mniejszy zapisz w licie wyjciowej
  • powtarzaj dopki jedna z list si nie skoczy
  • reszt dopisz na koniec wyjciowej listy

Algorytm scalania - przykład¶

[1, 6, 7] + [2, 3, 4, 5, 8, 9] --> [1]

[6, 7] + [2, 4, 5, 8, 9] --> [1, 2]

[6, 7] + [3, 4, 5, 8, 9] --> [1, 2, 3]

[6, 7] + [4, 5, 8, 9] --> [1, 2, 3, 4]

[6, 7] + [5, 8, 9] --> [1, 2, 3, 4, 5]

[6, 7] + [8, 9] --> [1, 2, 3, 4, 5, 6]

[7] + [8, 9] --> [1, 2, 3, 4, 5, 6, 7]

[] + [8, 9] --> [1, 2, 3, 4, 5, 6, 7, 8, 9]

def merge(A, B):
    """Scala posortowane listy zachowujc porzdek elementw."""
    result = [] # scalona lista
    while len(A) and len(B): # dopoki obie maj elementy
        # dodaj mniejszy element (usuwajc go jednoczenie z listy)
        if A[0] < B[0]:
            result.append(A.pop(0))
        else:
            result.append(B.pop(0))
    
    result.extend(A) # dodaj reszt 
    result.extend(B) # (jeli zostaa)
        
    return result

Złożoność mergesort¶

  • $\log_2 n + 1$ poziomw
  • na kadym $n$ operacji
  • zoono czasowa -> $O(n\cdot\log n)$
  • zoono pamiciowa -> $O(n)$

Sortowanie szybkie (quicksort)¶

  • wybierz element rozdzielajcy (pivot)
  • elementy nie wiksze przenie "na lewo", reszt "na prawo"
  • powtrz rekurencyjnie dla czci "lewej" i "prawej"

Sortowanie szybkie - przykład¶

[1, 4, 2, 5, 6, 3, 9, 7, 8] -> [1, 4, 2, 5, 3, 6, 9, 7, 8]

[1, 4, **2**, 5, 3] -> [1, 2, 4, 5, 3]

[4, **5**, 3] -> [4, 3, 5]

[4, **3**] -> [3, 4]

[9, **7**, 8] -> [7, 9, 8]

[**9**, 8] -> [8, 9]

Sortowanie szybkie - pivot¶

  • kluczowy jest wybr elementu rozdzielajcego
  • przecitnie, zoono algorytmu $O(n\cdot\log n)$, gdy dzielimy dane na mniej wicej dwa rwne podzbiory
  • gdy pivot jest wartoci skrajn, to mamy jeden podzbir $1$-elementowy i drugi $(n-1)$-elementowy; wtedy zoono algorytmu $O(n^2)$

pivot - wybór¶

  • stay element: brzeg lub rodek -> kiepski wybr
  • losowy -> teoretycznie kiepski, ale w praktyce si sprawdza
  • mediana -> dobry wybr, ale kosztowny
  • mediana z trzech -> mediana z pierwszego, ostatniego i rodkowego

# "jednolinijkowy if"- niezalecany (na potrzeby slajdu)
def quicksort(lst):
    """Sortuje podan list."""
    if not len(lst): return lst # nie ma co sortowa
    
    left, middle, right = [], [], [] # mniejsze, rwne, wiksze
    
    pivot = lst[0] # kiepski wybr
    
    for x in lst:
        if x < pivot: left.append(x)
        elif x > pivot: right.append(x)
        else: middle.append(x)
            
    # wywoaj rekurencyjnie dla "lewej" i "prawej" czci
    return quicksort(left) + middle + quicksort(right)

def bubblesort(lst):
    """Sortuje podan list."""
    n = len(lst) # liczba elementw
    while n > 0:
        new_n = 0 # miejsce ostatniej zmiany
        for i in range(n-1): # ptla od 0 do n-2
            if lst[i] > lst[i+1]: # porzdek zaburzony
                lst[i], lst[i+1] = lst[i+1], lst[i] # podmie
                new_n = i + 1 # indeks ostatniej zmiany
        n = new_n # ostatnie przejcie -> reszta musi by na miejscu
        
def merge(A, B):
    """Scala posortowane listy zachowujc porzdek elementw."""
    result = [] # scalona lista
    while len(A) and len(B): # dopoki obie maj elementy
        # dodaj mniejszy element (usuwajc go jednoczenie z listy)
        if A[0] < B[0]:
            result.append(A.pop(0))
        else:
            result.append(B.pop(0))
    
    result.extend(A) # dodaj reszt 
    result.extend(B) # (jeli zostaa)
        
    return result

def mergesort(lst):
    """Sortuje podan list."""
    if len(lst) < 2: # lista jednoelementowa
        return lst   # nie wymaga sortowania
    
    middle = len(lst) // 2 # rodek listy
    
    left = mergesort(lst[:middle])  # dziel na dwie czci
    right = mergesort(lst[middle:]) # i wywoaj merge_sort (rekurencja)
    
    return merge(left, right) # scal dwie listy

def quicksort(lst):
    """Sortuje podan list."""
    if not len(lst): # nie ma co sortowa
        return lst
    
    left = []   # elementy mniejsze
    middle = [] # elementy rwne
    right = []  # elementy wiksze
    
    pivot = lst[0] # kiepski wybr
    
    for x in lst:
        if x < pivot: left.append(x)
        elif x > pivot: right.append(x)
        else: middle.append(x)
            
    # wywoaj rekurencyjnie dla "lewej" i "prawej" czci
    return quicksort(left) + middle + quicksort(right)
